# style-book

CSS でできるあるある実装をまとめてみた，ほしょの遊び場

## 各実装の確認方法

各ディレクトリに`index.html`が存在するので，それをプレビューして確認してください．

## 自分でも作ってみたい場合

`make-style-dir.sh`というシェルスクリプトがあるので，そちらを使ってスタイルしたいディレクトリを作成してください．

```
source make-style-dir.sh 作成するディレクトリ名
```
